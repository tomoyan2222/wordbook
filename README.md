# README

Our,wordbook

アプリ概要: アプリ内で暗記用の英単語帳を作成、共有、閲覧できるアプリです。学生がテスト前に教科書の中から、英単語を自身で抜粋し、一人一人自分用の単語帳を作成しているシーンがあったため、誰かがひとつ英単語帳を作成すれば、共有するためのアプリと想定しています。
